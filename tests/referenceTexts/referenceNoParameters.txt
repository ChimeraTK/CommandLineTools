
mtca4u command line tools, version IRRELEVANT

Available commands:
	device_info  	Prints the register list of a device
	help         	Print the help text
	info         	Prints all devices
	read         	Read data from board
	read_dma_raw 	Read raw 32 bit values from DMA registers without fixed point conversion
	read_seq     	Get demultiplexed data sequences from a memory region (containing muxed data sequences)
	register_info	Prints the info of a register
	register_size	Prints the size of a register
	version      	Print the tool's version
	write        	Write data to board

For details on a command, run mtca4u command --help
